# W5 D5 Fri — Final Adversarial Review PR (Gate Day)

> **Mode: Gate day, not war-room.** Today IS the W5 assessment. Replaces v1's Exit Technical Exam — same security + failure-sim rigor, framed as a culminating PR review rather than test-taking. **This is the cohort gate** — score determines W6 entry calibration. No standard pre-session today; you read this brief + the rubric.

## What you're shipping today + why

Two PRs per pair, both scored. **Pair-project PR** (`phase2/aiops-hardening` branch) = your pair-graded surface. **`acquire-gov` PR** (`phase2/w5-shared-hardening` branch) = the shared brownfield surface where cohort-wide fixes (Item 3 circuit breaker on the shared evaluation-service) land. Both score. Both get codex review at Full strictness.

The PR is your defense. Codex provides the adversarial half. The instructor walks each PR live; codex findings land on the projector as they come in; your pair accepts, defends, or negotiates each. You're not test-taking — you're defending live code. A finding you can argue back on is as valuable as a finding you fix on the spot.

## Required PR contents (rubric-enforced — see rubric file)

ALL of these must land in your PR before codex runs. A PR missing any of items 1–5 is rubric-failed before codex even reads it.

| # | Required artifact | Verifying surface |

| 1 | **Resilience4j circuit breaker on Item 3** (evaluation-service → solicitation-service hop) | Test demonstrating breaker open/closed/half-open under load |

| 7 | **Test coverage delta non-negative** | CI report attached to PR |

## What clears the cohort gate

A pair scoring **Developing or above (≥20 of 40)** clears the W5 gate. **Below threshold** triggers an instructor + pair conversation about remediation path before W6 capstone work begins. The gate is **not** binary fail/pass — it sets the scaffolding intensity for W6.

| Tier | Score | What it means for W6 |
|------|-------|----------------------|
| **Distinguished** | 35–40 | W6 capstone runs without instructor scaffolding |
| **Proficient** | 28–34 | W6 capstone with light scaffolding on weak dimensions |
| **Developing** | 20–27 | W6 capstone with explicit instructor pairing on weak dimensions; remediation ticket required W6 EOD |
| **Below threshold** | < 20 | Cohort gate not cleared; pair + instructor decide remediation path before W6 capstone runs |

## How to come in ready

- **PR descriptions complete** following `templates/codex-adversarial-review.md` — explicit links to Wed HITL #7 ADR, Thu worth-it ADRs, Fri OWASP mitigation plan
- **Tests committed for items 1 + 2** — fix isn't a fix until the test fails on main + passes on the branch

- **Both PRs ready** — pair-project + `acquire-gov`. Ignoring the `acquire-gov` PR forfeits the shared-brownfield rubric weight
- **Sleep**. The session is 5 hours of defense. Bring a clear head.
