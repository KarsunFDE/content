## 5. Glossary refresh (terms you'll hear next Mon)

- **Resilience4j** — JVM circuit-breaker / rate-limiter / bulkhead / retry library. SB 3.x-native via `resilience4j-spring-boot3`. Replaces Hystrix (legacy) in 2026 stacks.
- **W3C `traceparent`** — HTTP header `traceparent: 00-{trace-id}-{span-id}-{flags}`. Standard. Every service in acquire-gov must forward + emit this by W5 Fri.
- **OpenTelemetry SDK** — emits spans, metrics, logs. Vendor-neutral. Datadog AI ingests OTel directly.
- **Auto-remediation authority boundary (HITL #7)** — for each AIOps-detectable failure: what does the platform fix autonomously, what does it escalate, what does it page on. The W5 Wed artifact.
- **Audit-log race detector** — W5 W*ed* deliverable: detect when an audit event is expected but not written (matching the Item 2 race). Foundation for the HITL #7 decision.
- **Datadog Watchdog AI / LLM Observability** — W5 hands-on platform per D-060. Per /web-research 2026-05-23, current top AIOps-on-AI feature surface.
